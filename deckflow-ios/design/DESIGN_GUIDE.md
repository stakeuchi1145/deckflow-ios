**状態**

## 7. 画面設計

### ログイン画面

| 状態 | 説明 |
|------|------|
| **Default** | ID・パスワード未入力、ボタングレー（`#808080`） |
| **Active** | ID・パスワード入力済み、ボーダー青、ボタン青（`#2196F3`） |
| **Error** | ログイン失敗時、エラーメッセージ表示（`#E53935`） |
| **Loading** | ローディング時、グレーオーバーレイ + スピナー |

**レイアウト構成**
1. ロゴ画像（160×160px、中央配置）
2. セクション: ID ラベル + 入力フィールド
3. セクション: Password ラベル + 入力フィールド（目アイコン付き）
4. ログインボタン（中央配置）
5. エラーメッセージ（条件表示）

---

### ホーム画面

| 状態 | 説明 |
|------|------|
| **Empty** | カードなし、「カードが見つかりません。」中央表示 |
| **With Cards** | カードリスト表示、スクロール可能 |
| **Loading** | データ取得中、グレーオーバーレイ + スピナー |

**レイアウト構成**
1. ヘッダー: 「My Cards」タイトル + メニューアイコン
2. 検索バー（グレー背景、角丸10px）
3. ディバイダー
4. コンテンツエリア（カードリスト or 空状態メッセージ）
5. FABボタン（右下固定）
6. ディバイダー
7. タブバー

---

## 8. デザイン原則

- **背景色**: 全画面 `#FAFAFA` でコンテンツ（白カード）を浮かせる
- **アクセントカラー**: Material Design Blue (`#2196F3`) を一貫して使用
- **ボーダーレス**: 入力フィールドは角丸なし（四角）でシンプルに
- **フィードバック**: フォーカス時のボーダー変色・ボタン色変化で状態を明示
- **タッチターゲット**: FAB 52px、メニューボタン 44px を確保
